2007/9/28 ～ 10/8
ポーランド・スウェーデン
9/28 Narita-Warsaw
今年も9月末から旅行。

ミュンヘンで乗り継いでポーランドの空港に降りると
雨が降っており、気温も低い。これはこれで雰囲気が出ていて良い。
両替してチケット買ってバスと徒歩でホテルへ。物足りないほど順調。

9/29 Warsaw-Krakow
ワルシャワはあまり時間がなかったが、早起きして電車の時間まで精力的に歩き回った。
戦時中蜂起した市民の像や、街の中心にある無名戦士の墓など 戦争関連のモニュメントが目をひく。

紅葉が始まる季節レストラン

市民の蜂起無名戦士の墓

午後からクラクフへ向かう。 列車内でも治安が悪い場合があるので油断は禁物と聞いた気がするが、 音楽を聞き始めたら瞬時に熟睡してしまった。

クラクフはいかにもヨーロッパらしい街並みが広がってた美しい古都で、 ここを基点に数日過ごすことになる。空もすっかり晴れ上がり、街がにぎわっている。

広場王宮周辺

王宮の上から教会

大きな教会の中を歩き回っていたとき、突然パイプオルガンで荘厳な曲が流れ始め、 新郎新婦が入場してきたので呆気に取られた。
そして大勢の観光客たちの祝福の中、結婚式が執りおこなわれた。

この後、突然バッハの一節が馬車とウエディングドレス

9/30 Oswiecim, Krakow
クラクフ近くのもうひとつの世界遺産、アウシュビッツ収容所跡へバスで向かう。
言葉を失うような展示の数々。 旅先で何かを実際に目にすると、写真や文章からイメージしていたものとは違うリアリティを感じるものだが、ここは特にそれが顕著だった。

観光客の中に、白いトレーナー姿の修学旅行のような学生集団を見かけた。 掲げていた旗を見て気づいたが、彼らはイスラエルからの訪問者らしい。

入口イスラエルから

天気も良く周辺は緑に溢れた気持ちの良い場所で、過去を知らなければこのあたりも ピクニックでもしたくなりそうだったが、それがいっそう虚しく感じられる。

夕方はクラクフの城など歩いて回った。日曜なので特ににぎわっているようだ。

老若男女がスポーツ車を利用ドラゴンの像の足元

10/1 Krakow-Zakopane
さて。観光と直接関係ない話で文章量の多い一日である。
何度かためらったのだが、今日と明日は野山を巡るため車を借りてしまった。
10年前の教習所以来のマニュアル車かつ左ハンドルというチャレンジなので、不安を隠し切れない。
駐車禁止2回、進路変更違反1回、転回禁止違反1回、速度超過違反1回、安全運転義務違反1回、 通算得点26点の卓越した運転技術に全てを委ね、今イスカンダルへ飛び立つ。

マニュアル車自体の記憶を完全に喪失していたため、 クラッチレバーなどが左右逆でも違和感すら感じないという予想外の特典もあったが、やはり技術的には無理がある。
加えてこちらの交通事情や縮尺の貧弱な地図にも悩まされ、大混乱のドライブになった。

レンタカー屋の敷地を出てから、まだ直進しかしていないうちにいきなり危機を迎えた。
工事で進路が変更されていることに気づかず、そのまま突入した道が路面電車専用だったらしい。 やばいと思ったときには周りの車が消えて線路と電車だけになり、前方に駅が見えてきた。 外では線路工事の親父が腕を振り回してこちらに何かを叫んでいる。最初から激熱展開。
幸い後ろから電車が来ていなかったため強引に方向転換し、工事用のダンプに着いて行って一般道に復帰できた。
方向転換の際、マニュアル車ってどうやってバックするんだっけ、と考えたという。

その後も道を間違えたりして大幅に時間を食いつつ、街を脱出して郊外に出た。
しばらくして最初の目的地、ヴィエリチカの岩塩採掘場に到着。

ここは地下深くにある岩塩の洞窟で、ただの採掘場でなくすべてが岩塩で構成された上、 迷宮のように縦横無尽に入り組んでおり、地底の聖堂や湖などもある神秘的な空間になっている。 入場料が高かった。

すべて岩塩でできた地下聖堂このクリスタルも塩

その後、南の国境近くにある町ザコパネへ向かって100kmほど走る。
途中で雰囲気の良い木造教会が道端にポツンと立っているのを発見し、緊急停止。
自由なドライブの醍醐味。

爽やかにたたずむ教会教会の裏で

ザコパネに到着して案内所で適当なペンションを紹介してもらった。
疲れきっていたが、陽気な夜のザコパネを歩き回って落ち着いた。

10/2 Zakopane-Krakow
ザコパネはリゾート地で、日本の避暑地やスキー場の雰囲気とちょっと似ていた。 もう一日くらい滞在して登山などしたかった。慌しいスケジュールなのが悔やまれる。

あちこちから陽気な音楽が聞こえるどこまでが公園かわからない

あちこちの煙突から煙があがる山の上から

寄り道をしながら車を走らせ、クラクフに戻った。 運転二日目になり、道路にたくさんの三角印が書かれている場所は どうやら「止まれ」らしいことに今さら気づく。

主幹道路だったせいか車の流れがあまり途絶えず、路肩に駐車しやすい場所も少ない。 マニュアルに不慣れなため強引な運転もためらわれ、色々残念なドライブだった。

ドライブからの生還祝いにワインをたっぷり飲んで気分よく眠る。
今日は泊まったゲストハウスも行ったレストランも当たりで満足。

トルンは北方の落ち着いた街で、かつてコペルニクスが住んでいた。

10/4 Torun-Gdansk
夕方、北の海沿いの港町グダンスクに移動。

川沿いの風景が美しいポーランドの人は花が好き。駅や空港で
花を持って人を待つ姿が多く素敵だった

明後日、ここから船でスウェーデンに渡る予定だが、実は出発の一週間前くらいに予約していた船会社からメールが届いている。

あなたが乗る予定だった船は欠航になりましたよ

という、理由も謝罪もないシンプルな通告だったので、慌てて別便をとったりして スケジュール調整をしたのだった。 不安が残り船着場の地図も難解だったので、港のオフィスに行ってみることにした。

駅の窓口で港までの切符を買おうとしたら首を横にふられた。
窓口の人は英語を話せないらしいので、意味がわからず悩んでいたら、 後ろに並んでいた男性が英語で色々教えてくれた。
他の場所でもこうやって親切にしてくれた人がおり、ポーランドの人は暖かい印象がある。 これがエジプトだったら誰かが話しかけてきても100%疑ってかかるところだが。

さて、教えてもらった話によると船会社の地図に記されていた電車最寄駅は、 あろうことか廃線になっていることが判明した。
代わりに教えてもらったトラムは時間がかかり、着いたとき既にオフィスは 営業終了していた。適当な旅行らしい駄目な展開が続く。

完膚なきまでに廃線で笑った港近くの砂浜で気分転換

10/6 Gdansk-
ポーランド最終日になった。
夕方の出航前にオリーヴァという町に行った。 ここの教会のオルガンはかつて世界一の音色と言われたそうで、その演奏を聞いてきた。
重厚なものから柔らかいものまで、音色の幅を広く使った演奏を楽しむ。

オリーヴァの公園オルガン

そして夕方、港からストックホルムに向けて出航。出国審査で色々質問をされた。

日本からですか？ 一人で旅行ですか？ それはデンジャラスですね。

という言葉から察するに、あまり普通の旅行者が選ぶルートではないらしい。
パスポートに船のマークの査証が刻まれたのが初めてで嬉しい。

そして乗船。キャビンをとるかどうか聞かれたが、寝るだけなので一晩くらいならと ラウンジで過ごしてみることにした。
しかし夜は思った以上に寒く、装備不足で震えるつらい一夜になってしまった。 電源が取れたので、PC の発熱すらありがたい始末。
時間はあったので、ここまでの旅記はこの時に船上でまとめて書いた。

朝になっても曇り空だったので海も寒々しい。陸地を待つ。

陰鬱な暗雲の中、出航バルト海と琥珀

10/7 -Stockholm
船はストックホルムから1時間くらい離れたニーネスハムン港に到着。
何組かの旅行者と共に、港からしばらく歩いて電車の駅に到着。 こっちは廃線になっていなくて良かった。
ここからストックホルムに向かうまで、ずっと景色が良かった。

陸が見えると鳥が集まってきたニーネスハムン港周辺

ストックホルムに到着し、日が暮れるまでしばらく街を歩き回った。
空はまだ曇っており、かなり寒い。

ストックホルム:夕1ストックホルム:夕2

ストックホルムは、小さな島々をつないで水辺にきれいな街が作られており、 首都にも関わらず計画的に景観を保って街づくりされているのは感心する。

ポーランドと比べて人々の雰囲気もちょっと変わり都会風というか現代風になった。 また、物価が3倍くらいになり、食事や買物に躊躇するようになってしまった。 500mlの飲料が300円からでは長期滞在はつらそう。
