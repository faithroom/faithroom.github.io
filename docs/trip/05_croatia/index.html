旅記 No.5
2005/9/18 ～ 9/27
スロヴェニア・クロアチア
9/18 Narita-
仕事の谷間の暇な時間を見つけて旅行に行こうと計画し、二年ぶりの海外旅行に出発。

だが仕事の谷間を狙ったはずの目測は完璧に誤っており、深夜まで仕事の毎日を突然逃げ出す脱走劇になってしまった。あとは知らん。

出発の夜、成田近くに住む叔父夫婦と会った。
実は今回の出発前夜に突然髪を丸坊主にしていたため、二人とも驚いていた。
変態か破戒僧か宇宙人か。

出発。

9/19 Ljubljana
スロヴェニアに入国。
ユーロ国籍とそれ以外のゲートに分かれていたが、非ユーロのゲートは 自分だけだった。質疑も妙にしつこく、機械で念入りにパスポートをチェックされた。
（にわか坊主のためパスポートの写真と別人と化していたことが一因くさい）

降り立ったスロヴェニアは雨で気温8度。到着早々だが、寒さと雨のせいでちょっと気が滅入る。 予定変更してまず上着を買いに走った。

装備を整え、ようやく落ち着いたので旧市街を歩き回った。

9/20 Blejsko
朝からブレッド湖に向かった。
ここは有名な観光地で、一日滞在してゆっくり一周した。
天候はすぐれず軽く霧も出ていたが、それが返って幻想的な演出になっていた。

9/21 Ljubljana-Zagreb-Plitvicka
クロアチアへ向けて出発。
時刻表から30分くらい遅れてようやく電車がやってきた。動じない。

暇だから撮った車窓から

今度の国境はスムーズに通過し、クロアチアの首都ザグレブに到着した。
ザグレブは最終日に戻ってきて観光する予定のため、 今日はそのまますぐバスを乗り継いで、山奥の観光地プリトビツェへ向かう。

車窓から景色を眺めていると、田舎町はクロアチアもスロヴェニアものどかなものだが、 クロアチアには窓も屋根もないような破壊しつくされた廃屋も多く、戦禍を感じさせる。
広場で普通に戦車と大砲が鎮座していたのはむしろ笑ったが。

9/22 Plitvicka-Split
プリトビツェは有名な湖群公園で期待していたのだが、今日は雨のため魅力も半減。 晴れていれば湖の色がもっと神秘的に輝いているはず。
それでも大小さまざまな滝は圧巻だった。

滝と湖1滝と湖2

迷いの森。この写真好きだ

午後からバスでスプリットへ出発。
しかしバス停はただの雨よけしかなく、時刻表もないので不安になってくる。

バス停

不安なのであたりの人に話して情報集め。

赤い鼻の老人：「こういうちいさなえきにはじこくひょうがないんじゃ。ザダールゆきのバスはまだかのう」
オーストラリア人夫婦：「つぎのバスはいつくるかわからないんだ。ぼくたちはこのさむいなか40ぷんもまっているんだよ。アドベンチャーだ。」

不安が増しただけだった。その後、30分くらい待ったら救いのバスが来た。
4時間半バスに揺られ、着いたのは港町スプリット。天気も晴れ上がり、暖かい海風が心地よい。 陽気な港町の雰囲気はちょっとバルセロナを思い出す。
この先のルートを決めていなかったので、交通関係を予約して回った。

スプリット港宮殿跡

9/23 Split-Dublovnik
スプリットには1000年以上前の宮殿が残り、中身だけ入れ替えて現代の街を作っている。
貴重な遺跡をそのまま生活に使うとはスケールが大きい。

宮殿千年前と同じ光景？

市場良い色の海

昼からさらに南へ向けて再び長距離バス。
また頭をからっぽにして外を眺めていた。
海沿いにぽつぽつと、赤い屋根の連なる街がある。
観光客用でない実用の馬車とすれ違った。

国境が奇妙な形をしているため、途中30分ほどだけボスニア-ヘルツェゴビナを経由した。 と行っても、山道の途中なのであまり景色に変わりはない。チョコレートを買った。

刹那のボスニア-ヘルツェゴビナ

徐々に南下してきた旅もここが終着点、ドブロブニクに到着。
千と千尋に出てきた魔女のような老女が民宿の客引きをしていた。 条件は無難だったのでそこに行ってみた。
やや誇大広告はあったが、なかなか快適だし民家の雰囲気がわかるのは面白かった。

旦那のニコが作った酒をもらった。甘く強い、異様に黒い酒だった。 材料の単語がわからなかったが、多分果物だろう。ニワトリの血とかではないと思う。

9/24 Dublovnik
街を囲む城壁の上を 2時間くらいかけてゆっくり一周した。
青い海をバックに赤い屋根の古い街を眺めながら、ああ来て良かったと思える景色だった。

街は普通に機能しているため、そこで暮らす人々の日常生活の様子までもが 城壁からの観光の一部のようになっているのが失礼ながら面白い。
通りを挟んで家と家の間に渡した紐に洗濯物を干す様子など興味をひかれた。

シスターの庭仕事うちてえ

午後はボートですぐ近くのリゾート地、ロクルム島へ渡ってみた。
せっかくなので少し海につかったが、こちらのビーチは完全に岩場で、 みんな足を怪我しないのかと不思議な場所で泳いでいる。

いまいちだったので泳ぎは早々に切り上げて島を散策した。
あちこちに野性の孔雀がいた。

島へ渡るボートから孔雀親子

ロクルム島一本道

この街は強力に観光地化が進んでいる。
戦争時に街が炎に包まれている様子のビデオすら土産物屋で売っていた。たくましい。
自分勝手な好みとしては、日本人客と日本語広告が少ないうちの方が好きだ。

9/25 Dublovnik-JadranskoMore-Rijeka
魔女の館をあとにし、今日から折り返しザグレブへ戻る。
しかしこの街には電車は通っていないし、妙に細長い国なのでバスで戻るにはもう一度同じ道を戻るしかない。
そこで選んだルートは、以前から一度やってみたいと思っていた船旅。
切符はスプリットでとっておいた。ザグレブの西リエカまで22時間かけて海路で北上する。

出港海は青く

船とすれ違う上部デッキから

俺は自由だ夕方の寄港。前に訪れたスプリット

船旅は想像以上に良かった。
デッキを何時間もうろつきながら、きらきら光るアドリア海を眺めていた。
持ってきていた音楽を聞きながら、すれ違うヨットに手をふる。

ときどき海に黄色い丸いものが浮いている。クラゲのようだ。
今日の自分はあのクラゲと同じくらい何も思考していない。

9/26 Rijeka-Zagreb
夜明けに港町リエカに到着。電車に乗ってザグレブへ。
よくわからなかったが車掌の指示で二度も乗り換え、5時間かけて到着。
首都ザグレブはさすがに華やか。食べ歩きや買い物などしてふらつく。
