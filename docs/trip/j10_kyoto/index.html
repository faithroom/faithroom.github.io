和旅記 No.10
2020/3/14～3/16
京都
3/14

世界的にコロナ問題が広がっている時期、予定のない三連休があった。 どこも観光客が激減しているという話で、調べた限り衛生的にも平日の通勤よりマシに思えたので、 当日企画でいきなり京都旅行に出発。

新幹線は往復とも自由席にしてみたが前後左右誰も座っていないほど余裕。
初日は到着が夕方だったので、とりあえず近くの清水寺から東山・祇園のあたりを歩く。 人が少なく歩きやすかったので、30分ほど散歩しながら夕暮れを待ってみた。

清水寺夕暮れを待てる余裕

二寧坂も閑散と

どこを歩いても驚くほど観光客が少なくて楽だが、観光業が気の毒なレベル。 数少ない外国人観光客は強い意志を持って来てくれたのかなと、ちょっと頼もしい。

普段ならば予約なしでは厳しそうな、良い感じの居酒屋に入ってみた。 おいしかった。明日はどこへ行こうか調べながら、観光業に貢献するためよく飲んだ。

3/15

前から行ってみたかった伏見稲荷に朝一で行ってみた。 まったく予習不足で、大きめの神社くらいのものを想像していたのだが、ちょっとした山だった。 二日酔いで朝食も採らずに来たので躊躇ったが、出直すのも面倒なので山頂まで登ってみた。
山頂までの参道に朝日を浴びる千本鳥居が延々続き、実に見応えがある。人気があるのも納得。

門番山頂付近

朝食後、やや遠出して天橋立へ。
天橋立は景色としては特段美しいとは思えなかったが、自然に造成された不思議な地形や、 日本神話を遺した古人の様子を想像したりして楽しむ。 少し雨にも見舞われたが、往復 8km ほど気持ちよく歩いた。
展望台にピアノがあったので勇気を出して弾いてみたが、かなり残念な演奏で逃げるように去る。

天橋立天の橋を歩く

京都への戻りの電車の中で調べていたら、夜の伏見稲荷も味があって良さそうだった。 温泉でちょっと休んでから祇園で夕食後、地下鉄で再び伏見稲荷へ。
朝一度登って勝手がわかっているので危なげなかったが、鳥居を照らす光は薄暗くちょっと怖い。 暴漢よりもお化けよりも野生動物が怖い。 あちこちに注意書きがあり、時々遠くでイノシシか何かの鳴き声が聞こえる。

朝と趣が異なる中腹の夜景

神秘的な山歩きで満足。振り返ってみると実によく歩いた日だ。

3/16

昼過ぎまでどこかへ寄ってから帰ろうと思い、比叡山に向かうことにした。
駅のロッカーに荷物を置いていったが、ほぼ全てのロッカーが空いていた。

空きランプが並ぶ

京都駅から20分ほどで琵琶湖のほとり。ここから歩き始める。
ケーブルカーで登るのが一般的らしいが、行きは歩いて登ることにした。 早朝ということもあってか、延暦寺到着まで人っ子一人見かけなかった。昨日に続いて動物がちょっと怖い。

登山開始朝日に輝く苔と雪

朝日に輝く木々が美しく、歩いて暑くなってきたので上着を脱いだりしていたが、山の中腹頃から木の上にかかっていた雪がパラパラ落ちてくるようになり、 延暦寺に着く頃には完全に雪になった。標高差はそれほどではなかったが、ここまで様子が一変するとは。

趣ある雪の延暦寺をしばらく観光。自分が突いた鐘の音が響き渡るのをじっと聞き入る。
本堂は大改築工事中で外観はまったくわからなかったが、参拝はできた。

阿弥陀堂・東塔中央正面に琵琶湖があるらしい

それなりの服装で来ていたが、さすがにこの雪で寒すぎる。 震えながらケーブルカーで下山。

京都駅に戻って新幹線で帰宅。やはり空いていて楽だけど、この状態が続いても困る。

Fin.
