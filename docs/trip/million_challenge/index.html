番外編: 月間100万歩チャレンジ
序
旅行やハイキングでよく出歩くので、どのくらい歩いているか歩数をカウントしてみようと思い DQウォークをインストールしてみた。 その中に月間歩数チャレンジの項目があったので、昨年 11月にがんばってみて月間 50万歩を達成した。

2022/11 50万

普段通りの生活だけではまったく足りず、かなり積極的に出歩いてなんとかこの数字を出せた。
で、その下に悪ふざけとしか思えない空欄がある。

異常者向け

運動狂や暇な学生ならまだしも、通勤片道 1時間で働く残業付き中年腐れエンジニアには縁のない話と思って無視。 年末は歩数カウントもせず忘年会漬けで過ごした。

しかし本当に無理だろうかという未練は微かに残っており、色々考えてみると来月はちょうど好機に思えた。

  • 1月は31日まである。
  • 1月は東京の降水量が少ない。暑くない。(寒すぎる?)
  • 年始に旅行休みを取っており、仕事日が少ない。新年会以外予定も少ない
100万が無理だとしても50万を超える最高記録は狙ってみようかと思い、年明けから試してみることにした。 ざっくり構想したことは
  • 単純計算で毎日33000歩というふざけた数字が基準
  • まず体を壊さないこと。年をわきまえろ
  • 1/10 から出勤なので、それまでに稼いでおきたい
  • とはいえそんなに真剣な話ではないので、仕事や遊びの予定は優先する
  • なるべく雨や出張がないことを祈る
結果が見えない中で書き始めたので、突然ギブアップやコロナ感染とかでしょぼい終わり方をする可能性も大。さて…
1/1～1/3
正月。実家の親と初詣等々でかなり歩いた。それに加えて緑地の散策などしていたが、やはり一日 33000歩は厳しい。
1/4～1/6
ここで稼いでおくしかない過酷なボーナスステージ。
別ページ参照。2023 富士山一周
1/7～1/9
出勤が始まる前の最後の休暇だが 1/7 は昨日までの反動でダウン。

ジョギングは距離と疲労のわりに歩数が稼げないので微妙だが、ペースと歩幅をかなり落として休憩を挟みながら毎朝 10km ほど走ってみた。 一日の基準に達するにはそれプラス2万歩ほど必要で、やっぱり異常な企画。

ここまでの冬休み期間のデータをまとめた。富士山で多少稼げたが、この程度の余裕では仕事が始まってからが確実に苦しい。

冬休み期間

1/10～1/11
通常出勤日に入った。出勤途中に寄り道して公園を一周して +5000 くらい。帰宅後に 10km ジョギングで +15000 くらい。その他日中と合わせて 25000。 やはり全然足りないペースになっている。今日あと1時間歩いてから寝るとかも可能だけど、膝も痛んできたし無理は続かない気がする。

当初この企画は思考ゼロの体力ゲームの認識だったが、わりと戦略パズルゲームっぽい気がしてきた。 疲労を計算に入れながら一日のどこに歩数を詰め込めるかのプランで頭を悩ませている。やることないのか他に。

1/12～1/13
ジョギング後から膝が痛くなり、翌日には本格的に痛くなって歩きづらい。これはもう限界か。
