番外編: 月間100万歩チャレンジ
序
旅行やハイキングでよく出歩くので、どのくらい歩いているか歩数をカウントしてみようと思い DQウォークをインストールしてみた。 その中に月間歩数チャレンジの項目があり、昨年 11月に月間 50万歩を達成した。

2022/11 50万

普段通りの生活だけではまったく足りず、かなり積極的に出歩いてなんとかこの数字を出せた。
で、その下に悪ふざけとしか思えない空欄がある。

異常者向け

運動狂や暇な学生ならまだしも、通勤片道 1時間で働く残業付き中年エンジニアには論外な話と思って、ここで活動は終了。 年末は歩数カウントもせず忘年会の酒浸りで過ごした。

しかし本当に無理だろうかという思いは微かに残っており、色々考えてみると来月はちょうど好機に思えた。

  • 1月は31日まである。
  • 1月は東京の降水量が少ない。暑くない。(寒すぎる?)
  • 年始に旅行休みを取っており、出勤日が少ない
  • 今健康
100万が無理だとしても50万を超える最高記録は狙ってみようかと思い、年明けからチャレンジを始めることにした。 ざっくり構想したことは
  • 単純計算で毎日33000歩というふざけた数字が基準
  • まず体を壊さないこと。コロナ感染など即 Game over
  • 1/10 から出勤なので、それまでに稼いでおきたい
  • とはいえそんなに真剣な話ではないので、仕事や遊びの予定は優先する
  • あとはなるべく雨や出張がないことを祈る
結果が見えない中で書き始めたので、突然しょぼい終わり方をする可能性も大。
さて…
1/1～1/3
正月。実家の親と初詣等々でかなり歩いた。それに加えて緑地の散策などしていたが、やはり 33000歩/day は厳しい。
新年会の店まで家から 12km 歩いていった。
1/4～1/6
ここで稼いでおくしかない過酷なボーナスステージ。
別ページ参照。2023 富士山一周
1/7～1/9
本年の出勤が始まる前の最後の休暇だが 1/7 はダウン。
ジョギングは距離と疲労のわりに歩数が稼げないので微妙だが、ペースと歩幅を落として休憩を挟みながら毎朝 10km ほど走ってみた。 一日の基準に達するにはそれプラス2万歩ほど必要で、やっぱりナンセンスな企画。

ここまでの冬休み期間の状況を整理した。富士山で多少稼げたが、この程度の余裕では仕事が始まってからが確実に苦しい。
