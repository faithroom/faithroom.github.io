旅記 No.18
2013/10/17～10/20
台湾
10/17
台湾は二回目だし短期だし、楽で堕落けた旅行だったし、そもそも読者少ないし。
…で、書くかどうか迷ったけれど、最近長文を書くこともなくなってしまったので、日記代わりに書くことにした。

木曜の夜に会社を早退して羽田直行した。
普段の出勤とカバンすら同じで、帰宅の方向を間違えただけみたいな緊張感のない出発。

台湾に到着して地下鉄でホテルへ。学生がスマホで LINE やパズドラをやっている。

今回の旅行は仲間三人で飲んで判断力低下しているときに始まった企画だが、全員飛行機もホテルも別。 既に一人は台北に入っているが、一人の方が気楽なので合流は三日目だけにさせてもらった。よくわからない企画。

10/18
朝一で龍山寺に参拝した後、台北の街を西から東までふらついた。

龍山寺常香炉

街で食べ歩きしたり買物したりしながら無計画に過ごした。
日本でも行ったことのないマッサージも行ってみた。なんだこの平和な旅行。

二年前に来たときよりも街に日本語が増えた気がする。
旅行中、英語すらほとんど使わなかった。

少し雨がぱらついていたが、夜市で夕食。安くてうまい食べ物ばかりで満足。
酒がないのが残念だが、あまり落ち着いて飲めそうな環境でもない。

酒と現地人との会話を求め、ちょっと緊張しつつ日本人の多いエリアのスナックへ行ってみた。 仕事で台湾に来ている日本人客たちが日本語のカラオケを楽しんだりしている。
流暢な日本語を話す姐さんに、色々ローカルな話など聞けて面白かった。 東海岸の花蓮を強く薦められたので、次回台湾に来たら行ってみたい。

普段むなしく放っている Twitter にも旅行中はちょくちょく反応がある。 酔ってだらだらとスマホをいじりながら眠りに落ちた。

10/19
朝食後すぐ電車で東へ向かう。
1時間くらいで乗り換え、まず平渓線というローカル線沿いの観光。 現地人中心に観光客で満員の電車だったが、適当に調べた情報を頼りに無人駅で降りたら 自分一人でいきなり心細くなった。

線路を歩こうと思っていたが、でかでかと禁止と書かれていたので山道をハイキングがてら回り道した。 しかしこれがどうやら失敗。Google map と実際の道のギャップが大きく、寄ろうと思っていた滝に至る道が見つからず 諦めて次の街、十分へ。

無人駅。平渓線は行っちゃった十分入口。天燈が迎えてくれる

この街の雰囲気は独特で、街の中心を線路が走り、電車が来ない間はわいわいと人が立ち入って商店街の一部のようになっている。
寛容さのない最近の日本だったら炎上しそうな光景だと思ってしまった。

一応禁止電車が来たら避ける

十分では願い事を書いて上げる名物の小さな気球、天燈がフワフワと上がっていく。 願い事のジャンルによって色が違うらしく、カラフルが気球が次々と上がり幸せな気分になる光景。

なぜか線路で上げる夢を乗せて飛んでゆく

更に電車で奥地へ行き、田舎町を歩く。
他の街でも天燈が上がっていたが、やはり十分が一番有名のようだ。

駅破壊中なんとなく昔の日本を感じる

平渓線を引き返し、バスに乗り換えて仲間との集合場所、九份へ。
山間の小さな町に赤い提灯に飾られた店が並び、面白い街だった。 ひっそりとした味わいを期待していたが、有名な観光地でしかも土曜ということもあり、 商店街は歩けないくらいのひどい混雑ぶりだった。

ここで仲間の一人から、やっぱり九份へ行くのやめようかと思う、と言う目を疑う LINE 着信。 相談の結果来ることになったが、もう色々適当だ。

九份到着。山の上だが海まで見渡せるどこもこんな感じの混雑

有名な千と千尋のモデルになった(自称らしい。公式には発表されていないとか)阿妹茶酒館で待ち合わせ。 絵になる店だが混雑も MAX。

昼夜

茶店で再会したのち、少し街を散策してからタクシーで台北に戻る。 仲間の友人も合流し、6人で夕食。海老中心の食事でおいしい。二次会は餃子屋。

九份の料理屋の裏側二次会の餃子屋。怪しい闇の奥に

見たいところを見て食べて飲んで話して、充実の一日だった。

終電がない。夜の繁華街を歩いて帰る。

10/20
台湾は朝から露店が充実しているので朝食は外で食べた。毎日こうすれば良かった。

帰りの飛行機は午後なので、南の烏来にある滝を見に慌しく出発。 地下鉄の終点からバスに乗り換え更に終点まで。今日も現地人の観光客が多いようだった。
山道をえらく揺れるバスに乗って烏来到着。

淡く濁った川立派な豚に出迎えられ

温泉街のある渓谷の川に沿って奥地へ進む。
行きはおもちゃのようなトロッコに乗ってみた。カタコトと楽しい。

小さなトロッコ渓谷

日本でも奥多摩あたりで見たような景色だと思ったが、よく見ると植物が亜熱帯風だったり 日本で見ないような鳥や虫がいたりと面白い。

滝に到着。写真のセンスがないのでまるで迫力がないが、実物はわりと壮観な瀑布だった。本当。

烏来の滝。壮観川辺でたたずむ

滝を眺めてリフレッシュしたので引き返し、バスと地下鉄を乗り継いで空港へ。

早々にチェックインを済ませてから、歩いて街へ出て昼食と土産購入。空港が街のすぐ横なので遊びやすい。 土産は有名店のパイナップルケーキで好評だった。

日本は寒そうなので空港で着替えながら、短い休暇を回想する。
台湾だと仕事を一日休んだだけでたっぷり遊べるものだと思った。また来たい。

Fin.
