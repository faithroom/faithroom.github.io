和旅記 No.12
2022/7/9 ～ 7/11
長崎
7/9 軍艦島～佐世保
唯一行ったことのない県、長崎に行くことにした。2泊3日で車の旅。
空港で車を借りてまず軍艦島へのツアーに参加したが、波が高くて上陸できなかった。 でも代わりに島の周りを２周しながら詳しい解説を聞かせてくれる。 廃墟をじっくり見たかったが、上陸しても自由に歩けるわけではないので外から全体を眺めるのもわりと良いかもしれない。

軍艦のシルエット軍艦島

あまり時間がないのですぐ長崎市を抜けて佐世保へ。
高速を降り、高台から九十九島を眺めようとスポットを探した。夕陽の沈む九十九島はさすが美しい。

九十九島陽が落ちる

ドローンを持ってきたので飛ばしてみた。まあ敢えて上空からでなくても良い景色だったのだが。 近くで別の老人も飛ばしていた。東京と違って気兼ねなく飛ばせる場所が多くて良い。

ドローン撮影

今日は丘の上のホテルで景色が良く、九十九島や佐世保港が見えた。 普通の港とは違い、巨大なクレーンが並ぶ造船所は遠くから見ても迫力がある。

7/10 平戸～生月島
本日のテーマは隠れキリシタンの島巡り。
朝一で平戸に渡り、しばらく町を歩きまわる。日差しが強く、肌が痛いほど暑い。 城の上からオランダ商館跡を眺めながら当時の貿易の賑わいなど想像する。

平戸の街並み平戸城とトンビ

丘の上の教会に行ってみる。趣は違うが、寺と教会が並ぶ通りはマレーシアを思い出す。
モダンなようなおもちゃのような不思議な色とデザインの教会。 事前情報から想像はしていたが、衛生上の理由により中には入れず。

ザビエル教会

次は平戸島の中央付近にある紐差教会。立派な教会だったがここも入れなかった。
でも窓が空いていたので中を見たらカラフルでメルヘンチックな作りが素敵だった。

紐差教会窓が開いていたので

平戸島からさらに橋を渡って生月島へ。昔好きだったアニメに出てきた島だが、当時はどこにあるかも知らなかった。
カーナビがなければ辿り着けない細道を登って教会へ。落ち着いた素敵な教会だったがやはり中には入れず。 あちこちにひまわりが咲いていてのどかな島。

生月島のひまわりの咲く教会

このあたりの墓地では、日本のオーソドックスな形の墓石に金色の文字で名が刻まれ、上に十字架が乗っているものをよく見かける。 独特の歴史と文化を思わせる。

生月島の北の岬は柱状の岩が連なる面白い景色が見られると知って行ってみた。
自然にできたと思えない不思議な地形で、空から眺めた島と海も美しかった。

塩俵の断崖美しい島だった

途中美しいビーチもあったが、時間が厳しくなってきたので泳ぐのは断念。
少ない日程で長崎一周プランを組んだため南端の雲仙まで長距離ドライブ。 高速道路は味気ないが降りると田園風景がのどか。

7/11 雲仙
朝食前に地獄巡り。硫黄と湯けむりで面白い景色だが、何かの間違いで突然危険な状態になったりしないのだろうか。
温泉も楽しんだが熱すぎた。時節柄か季節柄か客は少なかった。

地獄でくつろぐ猫朝の地獄めぐり

午後の飛行機まで何をしようか昨晩考え、普段の趣味の延長で登山にした。 ロープウェーがあるのでその先1時間程度で普賢山頂まで登れそうで、普段着しかなかったが何とかなりそうだった。 実際登り始めたら道が細く虫が多すぎてなかなか苦しく、引き返そうかとも思ったががんばって進んでみた。
道はだんだん良くなり、ほぼ誰にも会わずに無人の山頂に着いた。

みずみずしい山道雲仙普賢岳

帰りはロープウェーを使わない道を歩いて下ったが、こちらは大変快適な林道だったのでこちらで登れば良かった。

下山して公衆浴場で汗を流し、海の見える素敵な料理屋で最後のランチをとってから空港へ。 持ってきた水着は使わず、逆に登山は普段着しかなく、装備は色々失敗だったが充実した良い旅行だった。

Fin.
