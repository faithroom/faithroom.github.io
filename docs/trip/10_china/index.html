旅記 No.10
2008/5/4 ～ 5/7
中国
5/3 北京
連休を利用して北京への短期旅行に行ってきた。
空港から一人バスで街に向かうのはいつものごとくだが、 今回は留学中の友人 J の家に泊まり、案内もしてもらうという楽な道中である。
バスを降りてから J の家を目指したが、地図を見て想像したよりも遠いのは道と道の間隔が広いからだろうか。

さて、今回二つ返事で滞在を快諾してくれた J だが、目論見と外れて 明日提出の卒論が業火炎上中だった。 下手をすると観光は全部一人になり、自分は卒業の邪魔をしに来ただけの旅になるかもしれない。 そんなわくわくする状況を聞いている最中にも教授から電話で論文修正の指示があったりして、危機感が伝わってくる。

鎮痛な面持ちで教授と最終調整に向かう J を置いて、一人で市内の散歩に出かけた。
イメージどおり自転車も歩行者も多いが、そこに自動車がありえない勢いで突っ込んできて蹴散らしていく。 近年、自転車文化から車に移行しつつあるらしいが、まだ交通マナーに問題がありそうだ。

徒歩30分ほどで北京大学に到着した。
到着までに二三回車にはねられそうになった。と言うか一回かすった。

北京大学構内1

構内2電気街
ブランドの漢字名が面白かった

大学内は中国風のたたずまいの建物が多く落ち着いた雰囲気だった。
帰りは市内の電気街など覗いてから夕食。うまい。

その後、論文の大規模修正のため J が悲鳴を上げているのを横目に、だらだらテレビを見ながら寝る。 テレビでは海外用の NHK ニュースをやっていた。
でも中国に都合が悪い報道はカットされるらしい。 J によると、チベット系のニュースが映りかけた後に暗転したという。 つまり、あらかじめ検閲して編集しているのでなくリアルタイムで慌てて切り替えているということだろう。 なんというあからさまな情報統制。

5/5 王府井, 紫禁城
10秒に1回くらいのペースでやばいやばいと繰り返す徹夜明けの j を置いて、一人で市の中心部へ遊びに行った。
王府井という中心街で色々な店をのぞいた。怪しげな売り物も多い。 裏の方の屋台では虫のサナギなども売っていたが、食べる勇気はない。

王府井小吃街天安門

天安門から壮大な故宮に入る。ここはとてつもなく広い上、迷路のように入り組んでおり、 それこそ自転車で走り回りたい。

故宮太和門レリーフの階段

名前忘れた九龍壁

日本文化の多くはこの国から来たのだと納得させられるように、建物の形状はわりと近いものがあるが、 何に中国らしさを感じるのかと言えば、特徴的なのは色使いだろうか。
赤はもちろん青や金など鮮やかな色をふんだんに使った建物や石細工などの工芸品に、中国らしさを感じた。 現代の中国でも濃いピンクなどのやたら派手な服を着た人を見かけると、何か通じるものがあるような。

景山公園から。雲でなくスモッグかなこの色使い

裏手の景山公園から紫禁城を俯瞰できる。外人から forbidden city はこっちかと聞かれ、 立ち入り禁止かと聞かれているのかと勘違いした。英語で紫禁城をそう呼ぶのか。

観光客は西洋系の人も割と多かった。最近世界的に中国とは険悪ムードなニュースが多いので、 観光に興味を持つ人が多いのを見ると喜ばしい。

ここで J と再会。論文は提出が延期になり、終了の目処もついたらしいので、一緒に街へ出て前祝として飲み食いする。
J の案内で毎夜食事は安くてうまいものを食べられ、満足だった。 日本の中華料理ではなかなか目にしない料理ばかりで、種類の豊富さに圧倒された。

帰りににぎやかな繁華街を避け、静かなバーに寄る。
何かと人生観のかみあわない J と駄目な会話を炸裂させながら飲む。

水で字を書く。やたらうまい。大変品の良いバーストリート

5/6 万里の長城
今回一番楽しみにしていた、万里の長城へ。
果てしなく連なる長城には、何箇所もの観光スポットがあるが、 J が同行できることになったので今回は J も行ったことのない、ややマイナーな山を選択。
彼の中国語に助けられつつ、5時間くらいかけて到着。

長城は果てしなく

虫のようなJ塵のような自分

歩いたコースは起伏が激しく、ところどころ整備もされていないため、かなり疲れた。
長城は尾根づたいにどこまでも続いていたが、そうやって視界に入るのも 全体のほんの一部に過ぎない。
山がかなり険しいため、ここまで攻めてこられる勇者ならば 長城があろうがなかろうが関係ない気がした。

起伏はかなり険しい長城崩壊を食い止めた

小さな砦のたびに、中ボスのように控えている物売りが飲料やTシャツを売っている。
こちらは相手をする余力もないが、毎日売りに来ている彼らのバイタリティには敬服する。 それほどしつこくなく、エジプトの追尾システムには及ばなかった。

最後はスライダーで湖の上を滑走して派手にフィナーレ。
何より機器の信頼性にスリルを感じる。

村に引き上げる物売りたち脱出

無事北京に戻れてから、最終夜の今日は北京ダックを食べた。 皮だけでなく全部丸ごと、二人で丸々一羽食す。 良い店に案内してもらい、量も味も大満足だった。

J が趣味でやっている二胡を聞かせてもらい、自分も弾かせてもらった。怪音波が出た。

5/7 頤和園（いわえん)
帰りの飛行機まで時間があったので、頤和園という庭園を観光。
バスを待つ間、中央分離帯で年配の男女が工事をしているのを見た。 実は工事ではなくオリンピックのための植林活動らしい。

植林市

バスを降り間違えてかなり歩いたが、市場なども見物できたのでラッキーだった。
そして頤和園に到着。西太后の権力がうかがえる壮大な庭園である。

頤和園1頤和園2

湖の向こうまで頤和園庭園内に作られた街

昼食で入った軽いレストランは、やはり安くて量が多い。
人件費が安いためか店員もむやみに多くて、手持ち無沙汰で人形が陳列されたようだった。

その後帰国。近いと楽だ。短いが良い旅行だった。

後日、土産に持っていった漫画を読みふけっていてほとんど修正しなかった論文を J は無事提出したらしい。堕落万歳。

Fin.
