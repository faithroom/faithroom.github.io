和旅記 No.6
2016/2/11～2/13
キツネ村>
2/11
雪が見たい。温泉に入りたい。そして何よりキツネが見たい。
という衝動で宮城のキツネ村へ。

キツネ村と言うのは、たくさんのキツネが放し飼いになっていて一緒に遊べる愉快なテーマパークらしい。 前に TV で知って俄然興味を持ったのだが、つい最近もネットで記事を見かけたりして注目度が増している様子だった。

向かう途中福島で食事をし、食べ過ぎたので近くにあった信夫山に登ってみたりと無計画な寄り道を経てから白石へ。 駅からタクシーで15分ほどの鎌谷温泉に宿を取った。周辺には何もない隠れ里のような場所。

蔵王が見える

目的のキツネ村まではタクシーか徒歩を考えていたが、明日は金曜なので運良くバスがあるらしい。

夜は温泉と酒と料理に沈み、用意していた作業が一切捗らない。

2/12
バスでキツネ村へ。今年は暖かいが、それでも上の方は雪が多かった。 途中蔵王の近くを通り、スキー場の案内も目にした。

キツネ村前入口。鳴き声が聞こえる

中に入るとキツネが入れられた檻が並んでいて、 その奥では広場にたくさんのキツネが走り回っている。なんだこの幸せな場所。

雪の上にいた丸いスポンジ。

茶黒

キツネには雪が似合うだろうと思い、旅行しづらい冬を敢えて選んで正解だった。 雪の中で思い思いに過ごしているフワフワのキツネたちの美しいこと。 多くは歩き回っていたが、ずっと眠っているのもいるし走り回っているのもいるし喧嘩しているのもいるし交尾しているのもいるし、自由な世界だった。

雪が似合う近づいてくるが途中で逃げる

餌場で諍いふくらみすぎ。やたら太い尻尾

キツネという動物をほとんど知らなかったけれど、観察していたら面白かった。
ここのキツネはかなり近づいても逃げないが、本当に触ると危ないので触れない。 寒さ対策の習性なのか、丸まって眠っているのが多い。 あまり鳴かないが、鳴くときはわりと甲高い鳥のような声で鳴いていて、よくあるコンコンという表現はあまりしっくり来なかった。

寒そうな暖かそうな画屋根の上なら暖かいのか

昔からキツネは狡賢いイメージで語られるが、顔を見ていると確かに時折狡そうな目を見せる。 見ていると犬猫より表情豊かな気がして面白かった。

弛緩怒っている。良い表情

穴掘り。頭隠して頭に葉っぱを乗せたい

蔵王をバックに犬のようにも見える

尻尾が犬じゃなかったきれいだ

ということで、キツネ村を満喫してきた。家の近所にあったら通い詰めてしまいそう。 アクセスしづらい場所だけど外国人も多かった。

天気が良く山も見たかったので、帰りは徒歩3時間ほどで雪山を眺めながら下った。 下の方は雪もなくてちょっと退屈。

ダムと蔵王橋が雪で封鎖状態に

宿に帰ってから仲居さんにもキツネ写真を見せて語ってしまった。

2/13
今回は本当にキツネ村メインの旅行だったので、前後は写真もなし。 今日は仙台を散歩して伊達政宗像など見ておいしいものを食べて帰った。

昨日雪の中でずっとうろうろしていたせいか風邪をひいたらしい。
でもまたいつか行く気がする。

Fin.
