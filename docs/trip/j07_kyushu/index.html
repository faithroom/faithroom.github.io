和旅記 No.7
2016/5/2～5/5
九州
5/2
例年ならばゴールデンウィークに国内旅行は混むので避けるけれど、 先日の大地震の影響で九州旅行のキャンセルが相次いでいると聞いて行くことにした。
宮崎空港で飛行機から電車の乗り換え10分というギリギリの計画だったけれど、 運良く間に合って昼過ぎにはもう高千穂に着いた。

高千穂渓は残念ながら地震の影響で渓流沿いの道はほとんど立ち入り禁止だったけれど、 さすが渓谷が美しい。橋の上から眺めると目がくらみそうな高さ。

高千穂上の橋から。通行禁止の小道が見える

高千穂峡から1時間ほどかけて、神話に出てくる天岩戸へ。 神社の前にある学校の名が岩戸小学校とかすごい響き。

天岩戸神社

神社の参拝だけで直接岩戸は見なかったが、その奥にある天岩戸を開かせるための作戦会議をした天安河原という場所も美しいところだった。

岩戸の奥の渓流をたどるここで宴会の調整をしていたらしい

今日は町で借りた自転車で移動しているが、初めての電動アシスト付きだった。 赤信号で停止中に何気なくペダルに足を置いただけで前に押し出してくれる油断のならない機能だったが、 きつい山道が多いので助かった。これはすごい。
調子に乗って夕方バッテリーが上がるまで渓流沿いを走りまくった。

どこまでも渓流

夜は地産の焼肉。美味。

5/3
今日はずっと雨らしいので、4:30 起床で雨が降る前に国見ケ丘に登ってきた。 秋冬が有名らしいが、この季節も新緑が混じって美しい。丘の上は飛ばされそうな強風。

国見ケ丘

果たして雨が降り始めたが、これから長時間の移動なのであまり問題なし。 読書などしながらバスと電車を乗りつぎ、大分経由で由布院に向かう。
到着したが土砂降りで歩くのもつらかったので、ホテルに直行して温泉。
温泉で溶けかけていたら雨が上がったので夕方から外に出た。最高に気持ち良い雨上がり。

雨上がりの由布院おだやかに

全方位から蛙の大合唱が聞こえるあぜ道を歩いて金鱗湖へ。

金鱗湖ガア

それにしても、観光客はそこそこいるけれど温泉は貸し切り状態だし、ゴールデンウィークとは思えない落ち着き。

5/4
朝の由布院を歩く。あちこちから湯気のようなものが上がっているのは全部温泉だろうか。

朝の由布院昨日と同じ場所。寝ていた

みんな散歩帰り道に足を止めた。良い絵

由布院は同僚に勧められて寄ることにしたのだけど、良いところだった。 もう一日くらいのんびりしたかった。

ローカル線でがたごとと博多方面に向かう。停車時間が長い駅では降りて空気を吸ったりと、のどかな旅路。 今日の予定は直前まで決めてなかったけれど、熊本まで足を延ばすことにした。

由布院でもいくつか石灯籠が倒れたりしているのを見たが、 熊本はそれ以上に町のあちこちで被害が目立っていた。 熊本城の城内に入れないのは知っていたが、実際に激しく崩れた城壁を見たらちょっと衝撃的だった。

熊本城。中に入れないのでここから崩れた城塞。痛々しかった

その瞬間を想像すると怖い

写真を撮るのもどうかと思ったけれど、何年後かに修復されたときにまた来て今日の写真と比べてみたい。

5/5
毎日食べ過ぎなので最終日も歩きまくって消費する。まず大宰府の参拝。
およそ学問とは無縁の生活を送っている自分が祈っても相手にされなさそうだけど、形だけ参拝。

大宰府巨大樹

やけに賑やかだったのは子供の日のイベントのためだった。
参道の商店街では看板に燕対策をしているところが多い。していない店は巣が多い。

やまやの看板に

近くにある山に登ってみた。 小高い丘程度の山だったが、ビニール袋片手に普段着で登るのはちょっと苦しかった。 途中にダムやら鳥居やらがあってなかなか楽しい山で、ピクニック客も多かった。

水瓶山の入口ピクニックコース

ということで九州旅行を楽しんできた。やはり時期のわりに観光客はどこも少ない気がした。 出発2日前に立てた雑な予定だったけれど、来て良かった。
国内で3泊以上は学生時代以来だったと思う。 今回周ったのは初めての県ばかり5つで、まだ行ったことがない県として残すは長崎と沖縄のみになった。いつか行くだろうか。

Fin.
