旅記 No.26
2023/09/10～09/17
南イタリア
9/10 出発
ここ数年控えていた海外旅行へ久しぶりに。 イタリアは三度目になるが、一度南の方に行ってみたかったので目標を定めた。 今回はナポリより南の地域のみだが調べてみると行きたいところが多く、移動方法などかなり悩みながら慌ただしい旅程を計画した。

ナポリへの直行便はないのでイスタンブール乗換。イスタンブール空港は夜明け前なのに混みあっていて活気がある。 飛行機から見るイスタンブールの夜景に旅行気分が高まる。

夜明け前のイスタンブール

9/11　ナポリ～アルベロベッロ
ナポリ着。長距離バスの出発時間まで余裕があるので海まで行ってみたが、港であまり開けた景色は拝めず。 ナポリの雑然とした街を抜けてターミナルへ。

バスの中で今夜の宿の管理人とメールのやり取りをして鍵の隠し場所など教えてもらった。 しばらく迷惑メールに分類されていて気付かず危なかった。 こちらが英語で送っても徹底してイタリア語で返事が来やがるので翻訳アプリがなかった苦しかった。

東岸の街バーリに到着。30分遅れたがこのくらいは織り込み済みの旅程なので問題なし。 ここから次のバスでアルベロベッロへ行く。待ち時間で明日の電車の乗り場を確認してチケットを購入しておいた。
バスには行先表示の電光表示板が搭載されたりして、昔に比べてインフラもずいぶん快適になったなあと思っていたら、 電光掲示板が壊れて手書きされたバスが到着した。さすがである。乗り口で確認して乗車し、ようやくアルベロベッロへ。

アルベロベッロは想像より整然とした陽気な観光地だった。 伝統的な三角屋根のトゥルッリが面白い。よく見ると造りも様々で、観光用の近代構造のものも多そう。

近くの良さげなレストランに行ったら開店は19:30だそうで出直した。 明日以降の観光地もこのくらいの開店時間が多く、全体的に日本より遅め。 ビールと太めの超アルデンテパスタに大満足だった。歴史文化と食事で観光レベル最高のイタリア。

街は夜も賑わってそうだったが、長旅の疲れと時差ぼけで早々に寝る。

アルベロベッロのトゥルッリモダンな青いクロスが映える

9/12 アルベロベッロ ～ マテーラ ～ バーリ
まだ活動時間がずれているのでアルベロベッロの夜明け前から散歩。猫が懐っこくて良い。 月明かりの三角屋根から猫が降りてくるメルヘンな夜明け。
日が昇ってくると、街を散策する観光客があちこちに。

肉眼で見た景色はこちらの方が近いこの色はカメラで創られた画

三角屋根群道は整備されている

トゥルッリに泊まってみた。快適影もかわいい

途中で荷物を預けて電車でマテーラへ向かう。列車内で自転車を抱えた親切な人がみんなに乗り換え方を教えてくれた。

マテーラに到着。サッシはエッシャーの絵のような複雑な遺跡のような構造で、入口の案内から気分が上がる。

サッシ。意外と迷わない立体構造の洞窟協会

鳩味のあるマテーラ

奥の教会まで歩き、非常に暑い。 地図を見ると崖の向こう側は徒歩では行けない場所だったので引き返そうかと思っていたら、下の川に吊橋を見つけてしまった。 どうやらハイキングコースがあるらしく、時間を計算して思い切って挑戦。

吊り橋コース

とにかく暑い岩山を早足で川まで降りてまた登る。水着姿で登っている男女もいた。 時間も体力も厳しいので途中で引き返そうか何度か迷ったが、なんとか対岸まで往復。 川から見上げる街も良かった。
マテーラは今回の旅行で当初の期待を一番越えた街かもしれない。日帰りなのが残念。

対岸の洞窟から渓谷から。吊り橋がなくても渡れた

バーリに戻った。コロナ以降なのか天性のものなのか、今日の B&B は最後まで誰とも顔を合わせない特殊な運営だった。 直前に SMS で入室手順などが送られてきたが、長文のため分割された上に順番も狂っており、ゲームの謎解きのようだった。
棺のような昇降機が要注意らしい。

ゲームのヒント棺式昇降機

レストランでオーダー量を誤り、食べきれずギブアップしたが持ち帰り用に包んでもらえた。

9/13　パレルモ
旅程上、今日はライアンエアーで移動。徹底効率化で格安かつ追加(罰則)金が多いことで有名なLCCらしくて不安があったが、 事前に入念に確認していたので問題はなかった。
チェックインカウンターのクローズが早いなあと思っていたら、カウンターの担当者がそのまま移動して搭乗手続き担当をしており効率的。

シチリア島のパレルモに到着。宿の人が感じ良く、いろいろ観光アドバイスをくれた。
活気がある街で食べ歩きしながら観光。イタリアに来てからどの街でも、タトゥー、喫煙者、電動キックボードがやたら多い。 日本でも増えつつある電動キックボードは老若男女にかなり活用されていて、走り方も怖い。

メインストリートを歩いていて、美しい十字路があると思ったら有名な場所らしい。 十字路の良さを写真に収めるのが難しく、迷った末に上を向けて撮影してみたり。

美しい交差点試行錯誤

パレルモ大聖堂やノルマンニ宮殿を巡った。 複雑な歴史や建築様式などに詳しければもっと感動できただろうにと思いつつも楽しんだ。

大聖堂内に近年亡くなったらしい司祭の慰霊廟のようなものがあったので何気なく経歴を読んだら、熱心な社会活動の末 最期はマフィアに殺されたと書いてあって不意にシチリアを感じた。

司祭の慰霊大聖堂の屋上

宮殿は宮殿自体より、圧巻のパラティーナ礼拝堂や発掘された食器類などの展示が面白かった。

パラティーナ礼拝堂モザイク画のアップ

ちょっと疲労がひどくなって少し昼寝。年には勝てない。いや昨日のハイキングで無理したせいか。

この裏が宿ゴッドファーザーに出てきた?

夜は宿の人が薦めてくれたレストランに行ってみた。満足度高い。

9/14　ランペドゥーザ
今回の旅行ではどこか離島にも行きたいと思って事前に調べ、ランペドゥーザ島に決めていた。 パレルモから南に 300km、プロペラ機で到着した南端のリゾート地、ランペドゥーザ。
空港から徒歩で近くの宿に寄り、昼食へ。観光客はみんな海に行っているらしく、メインストリートは閑散としている。

明るいランペドゥーザのランチ

明日はボートツアーに参加したかったが、目的のツアーは電話がつながらず、拠点まで行ってみても申し込みできず困った。 最終的には予約できたが、そんなことでグダグダと無駄に時間が経ってしまったので今日はビーチはやめてバスと徒歩で島内を周ってみた。

のどかな村や自然もあるかと期待していたのだが、基本的に延々荒野しかなく、なんでリゾート島に来てこんな修行みたいなことやっているんだという謎の午後になってしまった。
荒野をさまよって喉がカラカラになったが、水を買えるスーパーはどこも 16:30 オープンらしい。この情報誰かの役に立つだろうか。

崖から海を眺める砂漠か

荒野世界各地への道標が面白い

夜のメインストリートは人通りが多いのに、どこのレストランも暗く閑散としていて何かおかしい。 教会前に人が集まっていると思ったら、蝋燭を持った大勢の市民が海に向かって行列を作って歩き始め、何か宗教的なイベントがある日のようだった。 レストランもその間は営業中断しているようだったので、空腹に耐えながら他の観光客と共にしばらく行列を眺めてじっと待った。

吹き出すような花蝋燭を持って歩く人々

その後無事食事をして明日に備える。あとでこの儀式の正体を調べたが結局わからなかった。

9/15 ランペドゥーザ
期待のランペドゥーザ島一周ボートツアー。素晴らしい体験だった。
ポイントポイントで錨を下ろして停泊し、見どころを教えてもらってみんなで海に飛び込む。 ビーチで泳ぐより絶対こちらの方が楽しめる。
海底の岩質なども影響し、海と思えないような透き通るような水。この内側の島が昨日の荒野とは思えない。

ツアー船上飛んでいるように見える船が有名らしい

今回の旅行写真は色補正なし船が飛ぶ

シュノーケルを付けて魚にエサをやったり沈没船を探したりとよく泳いだ。 わりと年配の方たちも自分以上に元気に泳いで楽しんでいて魅力的な老人たちだった。将来自分もそうありたい。

場所を聞いて沈没船の錨を発見財宝が眠っていそう

水中での写真撮影はなかなかうまくいかず、ほぼ全部ボツで削除。事前に確認した通り、動画はそこそこ撮れたが。

ツアーはイタリア語なのでさっぱりだったが雰囲気で楽しんだ。 英語で話しかけてくれる人もいて、愉快な老姉妹は特によく相手をしてくれた。 各々の紙コップに名前を書くように言われ、老姉妹は名前のあとにハートマークを書いていた。

歴史文化と食事に加え、大自然も満喫できたイタリア。

こんな船内で盛り上がっている強力な老姉妹(海で喫煙中)

出発前、ランペドゥーザの情報を検索していたら、最近治安が悪化しているという話ばかりがヒットして本当に行くかちょっと迷った。
元々アフリカからの難民もしくは移民(これがどちらかというのも微妙らしい)がこの島に船で大勢やってくるのが問題になっていて、この夏はその人数が激増して島の住民数を越えたらしい。 ほとんどの日本人は名前も知らない島だろうが、狙ったようにこの旅行中のタイミングで NHK のニュースにもなっていた。

それでも予定変更せず来てみたのは、観光客からは最近でも満足そうな感想しか見られなかったこと、 危険情報を発信しているのはわりと政治的な思惑が強そうな人が多くて誇張気味に思えたこと、 かつて行こうかと思っていたけれどその後の事情で行きづらくなった国があり残念な思いをしたことが何度かあったのを思い出したこと、などなど。

若干の不安を感じながらも来てみたら、自分はまったく問題なかった。 ツアー船から移民収容施設が見えたので船長に聞いてみたら、ああ、問題になっているんだよと軽めの反応だった。 透き通る海を楽しんだだけでなく、そういう状況も含めて深く記憶に残る島だった。

「人身売買反対」平和な島でありますように

9/16 ポンペイ
飛行機の乗り継ぎで午前中にナポリに戻った。 想定外に荷物の追加料金が発生してしまった。ライアンエアー以外にも注意が必要だったが油断した…

ナポリの街でもぶらつこうかと思ったが、時間があったのでポンペイまで足をのばした。 四半世紀前に来たはずだがよく思い出せない。当時からは見違えるほど整備されている気はする。

遺跡でひたる遺跡を歩く

未曽有の大災害の被災地を好奇心で楽しく歩き周って良いものかとふと気になったが、2000年も経つとどんな惨劇もロマンに化ける感覚がある。
来たのが遅かったので閉園直前まで歩き、小雨の降る夕暮れの遺跡を最後に見られて感傷的になった。

遺跡の植栽も面白いポンペイに日が沈む

元凶の山旅行のエンディングは虹

久ぶりの海外旅行を堪能し、密度の濃い一週間だった。 日頃それなりに仕事を楽しんでいたはずだが、旅行中は驚くほど思い出さなかった。

1分のショートムービー。。
