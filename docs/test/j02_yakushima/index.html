和旅記 No.2
2010/8/11 ～ 8/13
屋久島
8/11 海
前から行きたかった屋久島に行ってきた。
会社の仲間の実家が近いのがきっかけで計画が始まり、現地合流することにした。

朝4時起きで、自転車→電車→電車→電車→飛行機→バス→船→車、で現地到着。 移動だけでえらく疲れたが、二泊三日の短期のため休む間もなく海に向かう。

今回はガイド付きで初めてのシュノーケリングをやってみた。 浜の近くでも驚くほど多種の魚や他の生物が見られて面白い。しばし没頭した。

キノコのような珊瑚小さい魚が群れている

新兵器のカメラ防水パックは反射で自分が何を撮っているかよく見えず、 ほとんどは勘で撮ったが、なかなか面白い写真も撮れていた。

ガイドの人はウミウシとかナマコとか、珍しいものを見つけて説明してくれる。
シュノーケリングは簡単で楽しい。ダイビングだと更に感動が違うのかもしれないが、今回はシュノーケリングで充分満足。

南国らしい色彩同行者。自分も同様の装備

魚を見て楽しんだあとは魚を食して楽しむ。小さな町だが良さげな料理屋が多くて迷った。 夕方になると山から涼しい風が吹き、散歩も気持ちよい。

何かなごむ島の夕暮れ

明日もまた4時起きのため健康的に寝た。

8/12 杉
今日は屋久島で有名な縄文杉へ向かうため、夜明け前に出発した。人智を越えた樹齢の妖怪のような木らしい。
休憩を入れて全部で往復8時間くらいの行程になった。 序盤は雨の中を大勢一列になり、緩やかなトロッコの線路上を歩いていく。

トロッコの線路が延々続くたまに登山客を蹴散らして走る

朝食の弁当を食べようと脇の草地に立ち入ったらすぐ注意された。自然保護が厳格だ。
広い場所に出てから弁当を食べていたら雨もあがり、ようやく本番といったところ。

神秘的な森根がからみあう

二時間ほどのトロッコ道が終わったところが中間付近で、ここから先は距離は短いが歩きにくく、 前がつかえるので時間がかかる。

水と苔が美しい。この山だけでなく島全体かもしれない。 巨大な切り株が面白い見所のひとつだった。中から空が見える雄大な切り株。

切り株の中から杉嵐

芝生でなく苔木の表面も苔が多く、緑色に光る

逆さ杉空を見上げる

そして目当ての縄文杉に到着。なんとまあ、いまいち感動がなかった。 えらい人ごみだったのと、途中の山道で他の立派な大木や瑞々しい鮮やかな景色を目にしてきたせいかもしれない。 とりあえずでかくて、横に人面みたいのがある妖怪の一種だった。

縄文杉

屋久島には鹿と猿が多い。 昨日の海のガイドの人が、屋久島の鹿や猿は表情が優しいと言っていたが、実際見たらわかった気がした。

鹿。人懐っこい猿の親子

下りは楽だが、疲れもあってボーッと歩く。
下山後、同行者の親戚から夕食に招いていただいた。日本の田舎らしいのどかな夕暮れにバーベキューを満喫した。 後ろでは親戚の悪ガキが、カマキリとバッタを同じ虫かごに入れて凄惨なバトルをプロモートしていた。

どうも旅行では似たようなパターンが多いが、今日も昼は鹿とたわむれて楽しみ、 夜は鹿を食べるという切り替えの早さだった。

民宿の巨大猫お邪魔した庭先で

8/13 山
よく旅行前には本や映画でイメージを高めるのだが、今回は屋久島の山をモデルにした「もののけ姫」を直前に見直してきた。
映画の取材後にもののけ姫の森と名づけられた森のある、白谷雲水峡に向かった。

昨日の登山に続いてハードだが、朝もやのかかった状態の山が見たくて早朝に出発。 早朝でないと帰りの船が間に合わないという理由もあったが。

山の入り口に着いた時点で土砂降り。多少の雨ならば問題ないが、沢を歩くには危険を感じるレベルだったので、 まず休憩所で弁当を食べて雨が弱まるのを待った。この作戦勝ちで、ほどなくして雨は弱まってきた。

水溢るる山ようやく雲が晴れ

朝もやのかかった山を川に沿って登っていく。
雨が降っているせいもあると思うが、山の中のいたるところに水場があり、 水溢るる森という印象だった。昨日の山以上に苔が美しい。

一応雨具は持ってきていたが、雨に濡れるより雨具の熱気の方がつらかったため、ほとんど使わなかった。

神秘的な森にみとれる

木の幹も緑岩も緑

本日の目的地は太鼓岩。
森の木々の隙間を縫うようにして険しい山道を上った果てに、突然開けて周囲の山々を見渡せる大岩がある。 ちょうど到着する頃に雨も完全に上がり、青空が広がった。滅多に味わえないような幸運と爽快感。

森だけでなくこの太鼓岩も、もののけ姫のモデルになった場所だと見た瞬間わかる。 「黙れ小僧」とか怒鳴っていた場所だ。

太鼓岩あと一歩で奈落。よく滑る

太鼓岩から登山は早朝が良い

帰りは木漏れ日に照らされた苔を眺めながら山を下る。
これから訪れる人は、もし一日しかなかったら縄文杉よりこちらを勧めたい。

昼食後、島から引き上げた。二泊三日の短期だったが、充実した良い旅行だった。
帰りは来たときと逆の長い道のり、バス→船→バス→飛行機→電車→電車→電車→ここで自転車の鍵をなくしていたため、徒歩。 最後で駄目なオチがついた。

Fin.
