和旅記 No.0
2006/08/20～08/21 - 富士登山
富士山へ
日曜の夜から月曜にかけて、会社の後輩を引っ張って富士山に登ってきた。
選んだルートは須走口。山頂までの行程はやや長いが、比較的人が少ないと聞いたので 迷わずここを選択した。
車で真っ暗闇の道を五合目まで登り、駐車場で登山準備。22時頃登山開始。
五合目
懐中電灯を頼りに進む。 頭に装着するタイプが便利らしいが、そんなもの買っても今回以降二度と使うまいと思い、 汎用性の高そうな普通の懐中電灯を買った。
阿呆な友人に薦められた こんな 軍事兵器みたいなやつを買わなくて良かった。

さすがに暗い

六合目
ここまでは難なく到着。
木が少ないので視界が開けている。暗くても思った以上に歩きやすかった。
七合目
もう半分以上の距離を来たが、歩きやすかったためあまり疲労はない。 辺りはかなり寒くなり、スキーウェアを着用。休憩して止まっていると寒くてつらい。
山歩き用の曲つめあわせを用意していたので聴き始めた。
八合目
急激につらくなってきた。と言っても体力的な疲れとはちょっと違う。
この付近で別ルートが合流するせいもあり、登山者が激増したせいだ。 行列になっていて自分のペースで登れない上、あちこちでツアーのコンダクターが 叫びまくっていて実にやかましい。

そして冷気と眠気が激しくなってきて状態がどんどん悪化していく。 歩いていると頭がふらつき、立ち止まると寒くてやっていられない。

振り返ってみれば、登山者達の懐中電灯の光が点々と。

蛍の光のように

九合目
気づかないうちに九合目を通過していた。
半分眠りに落ちた精神で、自分の体をマリオネットのように操って淡々と歩かせるようなよくわからない感覚。
眠い寒い眠い寒い眠い寒い眠い寒い眠い寒い眠い寒い眠い寒い眠い寒い。
山頂
ついに到着した山頂。イメージと違い、縁日のように店が並びごったがえす人。

5合目から 5時間半とペースが早かったため、日の出まで約1時間待つことになった。 少し眠ろうとしたが冷気が強力すぎてそれどころでなく、ふるえが止まらない。 と思ったら後輩はうずくまって眠っており、揺らしても起きない。
もっと揺らしたら起きた。生きていた。

経済の基本に忠実な 700円のカップラーメンを食してから、高台で日の出を待つ。 今度こそ寝たら死ぬ。

そして日は昇る。

待ち遠しい日の出

苦労が大きかった分、神々しい朝日に感動もひとしお。
感動を伝えようと携帯でメールを送りまくった。電波入るんだな…

後輩と黄泉比良坂？

地獄のような溶岩跡を巡る。

幻想的雲海の彼方に島のような山

こんな景色が見られるのですね一気に気温が上がってきた

下山
帰りのルートは行きと異なり、延々続く一本道の砂利道。 ズザズザと半分すべるように蛇行して駆け下りたら 3時間足らずで降りてこられた。

後輩は途中で足を痛めたとかで、後ろ向きに下りていた。楽なのかそれは？

砂利スキー

まとめ
普段のハイキングの延長で、日本一高い場所に登れれば満足という軽いコンセプトで計画したのだったが、 山頂の景色がこれほどすばらしいとは思わなかった。

寒さと眠さの波状攻撃はつらかったけれど、いつかまた行くかもしれません。
